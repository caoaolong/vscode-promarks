# Changelog

## 0.0.3

- 标签系统
  - 为项目新增多标签支持（可选已有标签或新建逗号分隔标签）
  - 顶部新增“按标签过滤”，可多选或清除筛选
  - 新增“Sort by: Tag”按标签分组展示，未打标项归入 “Untagged”
  - 卡片右下角新增 “Tags” 快捷入口，快速单独编辑标签
  - 标签颜色：基于标签文本生成稳定色相的圆角 Chip，适配 Light/Dark 主题
  - 标签展示最多 3 个，多余以 “+N” 汇总
- 卡片布局优化
  - 减小网格与内边距、卡片最小高度、自适应文本
  - 备注两行省略、整体信息密度优化，避免内容溢出

## 0.0.2

- 新增 SSH 远程项目：支持添加并从列表打开 Remote(SSH) 项目
- 新增项目展示分组/排序切换：按语言、按时间（最近一周/更早）、按位置（本地/远程）
- 新增排序模式选择下拉框，记住上次选择
- 新增本地/SSH 两类项目的新增入口卡片

## 0.0.1

- 初始版本：实现空工作区自动展示 Project Manager 页面
- 网格布局项目卡片：名称、语言与图标、上次打开时间
- 支持项目管理：新增项目、删除项目、编辑备注与语言类型
- 点击卡片打开项目目录（在当前窗口打开）
- 支持插件与页面图标（logo.png）
- 新建后是否自动打开项目：页面底部复选框控制（全局持久化）
